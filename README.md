# Start el-api localy (Ubuntu 16.0x)

***install last LTS version of node***
```sh
sudo apt-get update
sudo apt-get install nodejs
```

***install TypeScript last LTS version***

***set env settings in /app/config/***
```sh
mongoDbUr...
etc....
```

***start the project***
```sh
npm run watch-ts
npm run start 
```
